
## Desafio MBR

Bem-vindo! Este projeto é um site de quiz simplificado, desenvolvido com HTML, CSS e um pouco de JavaScript. A proposta é oferecer uma experiência minimalista, permitindo aos usuários inserir seus nomes e navegar entre as telas.

## Estrutura do Projeto

- `index.html`: Página inicial com a inserção do nome.
- `quiz.html`: Página principal do quiz.
- `score.html`: Página de pontuação do quiz.
- `styles`: Estilos para as páginas HTML.
- `home.js`: Funcionalidades JavaScript da primeira tela para interação e navegação.
- `quiz.js`: Funcionalidades JavaScript da segunda tela para interação e navegação.


## Funcionalidades

- **Inserir Nome:** Os usuários podem inserir seus nomes na página inicial.
- **Mudar de Tela:** Navegação simples entre a página inicial e a página do quiz.

## Como Usar

1. Abra o arquivo `index.html` em seu navegador.
2. Insira seu nome na tela inicial.
3. Clique no botão para acessar as telas seguintes.
